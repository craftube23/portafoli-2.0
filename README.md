## 🎮 System Overview
Este es mi portafolio profesional diseñado con una estética **HUD/Gamer**. El objetivo es fusionar la armonía de la música con la precisión del código, creando una experiencia visual de alto impacto.

* **Entorno de desarrollo:** Desarrollado íntegramente sobre **Bazzite OS** (Fedora based).
* **Estética:** Inspirada en interfaces de escaneo, neón y *scribble art*.
* **Performance:** Optimizado para una navegación fluida con animaciones de radar y glitch.

---

## 🚀 Tecnologías Core
* **React / Next.js:** Arquitectura de componentes "use client".
* **Tailwind CSS:** Estilizado mediante variables de neón personalizadas.
* **Lucide React:** Iconografía minimalista y funcional.
* **TypeScript:** Tipado fuerte para un código más robusto.
